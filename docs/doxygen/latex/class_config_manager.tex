\doxysubsection{Detailed Description}
The \doxylink{class_config_manager}{Config\+Manager} class. 

Gestiona la configuración de la aplicación desde un archivo config.\+ini. Aplica el patrón Singleton para asegurar que solo existe una instancia accesible globalmente.
