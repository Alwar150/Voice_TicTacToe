Create or update a configuration by parsing slightly extended JSON.

This function parses a JSON object in non-\/strict mode to produce a \doxylink{structps__config__t}{ps\+\_\+config\+\_\+t}. Configuration parameters are given {\itshape without} a leading dash, and do not need to be quoted, nor does the object need to be enclosed in curly braces, nor are commas necessary between key/value pairs. Basically, it\textquotesingle{}s degenerate YAML. So, for example, this is accepted\+: \begin{DoxyVerb}hmm: fr-fr
samprate: 8000
