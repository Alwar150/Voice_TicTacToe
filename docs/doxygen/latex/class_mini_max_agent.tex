\doxysubsection{Detailed Description}
The \doxylink{class_mini_max_agent}{Mini\+Max\+Agent} class handles all the logic of picking a cell based on the Mini\+Max Algorithm. 

It simulates all possible plays taking turns simulating an AI turn then a \doxylink{class_player}{Player} turn until a final state is reached, then it scores the final state by giving a negative value if the \doxylink{class_player}{Player} wins and a positive value if it wins, then backtracks picking the minimum score if it is the player turn and the max score if its the AI turn.
